The User Profile & Dashboard Fields Control plugin makes it simple to manage user profile and dashboard fields.

== Description ==
The User Profile & Dashboard Fields Control plugin allows administrators to customize which profile and dashboard fields are visible based on user roles. Streamline user management and enhance the experience by tailoring settings to meet specific needs.

== Installation ==
1. Upload the `user-profile-dashboard-fields-control` folder to the `/wp-content/plugins/` directory.
2. Activate the plugin through the `Plugins` menu in WordPress.
3. Configure plugin settings under `User Profile & Dashboard Fields` in the WordPress admin menu.

== Frequently Asked Questions ==
= How do I configure the plugin? =
Navigate to `User Profile & Dashboard Fields` in the WordPress admin menu to access and configure the plugin settings.

= Can I control which fields are hidden based on user roles? =
Yes, the plugin allows you to specify which fields are visible or hidden for different user roles.

= Is the plugin compatible with all WordPress themes? =
The User Profile & Dashboard Fields Control plugin is designed to work with most WordPress themes. However, compatibility may vary based on theme customizations.
